# RxStream Analytics Platform

## Project Overview

- **Description:** Real-time sensor monitoring dashboard with threshold-based alerting, configurable rule engine, and dynamic Chart.js visualizations using RxJS reactive streams.

- **Problem & Solution:** Built RxJS-based reactive streams with customizable rule engine to solve lack of real-time responsiveness and configurable alerting in traditional sensor monitoring systems.

- **Tech Stack:** Angular 21, TypeScript, RxJS, Chart.js (ng2-charts), HTML5/CSS3 for real-time data simulation and reactive visualization.
